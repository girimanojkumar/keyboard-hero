>># How to use 
>>* Left Alt + h = control + left arrow
>>* Left Alt + l = control + right arrow
>>* Left Alt + i = control + up arrow
>>* Left Alt + j = control + down arrow
>>
>>* Left Alt + Space = control +  right >arrow = go one word right
>>* Right Alt + Space = control +  left >arrow = go one word left
>>
>>* Alt + shift + h = control + shift + >right arrow
>>* Alt + shift + l = control + shift + left >arrow
>>* Alt + shift + i = control + shift + up >arrow
>>* Alt + shift + j = control + shift + down >arrow
>>
>>* Alt + [ = control + home
>>* Alt + ] = control + end
>>* Alt + shift + [ = control + shift + home
>>* Alt + shift + ] = control + shift + end
>>
>>* Right Alt + i = alt + up arrow = move >line up in vscode
>>* Right Alt + j = alt + up arrow = move >line down in vscode
>>
>>* Alt + f = control + shift + d = add >selection to next find match
>>* Alt + d = control + d = duplicate >selection in vscode
>>* Alt + / = control + / = commenting line >up in vscode
>>
>>* Alt + c = control + c = copy
>>* Alt + v = control + v = paste
>>
>>* Alt + v = control + v = paste
>>* Alt + 8 = control + z
>>* Alt + 9 = control + shift + z
